# Satisfactory 1.0 — Build Guide (companion)

> **The live, exact step-by-step is the ✅ Do Next tab in `factory-network.html`.**
> It numbers every step HUB → launch, with the kit to bring, belt/pipe routing,
> per-phase tech unlocks, power, utilities, and check-offs. This file is the
> prose overview; the app is the checklist.

Read **README.md** first for the model (6 material districts + ★ Project
Assembly, top-right desert, above-ground, MAX overclock).

## Build order in one breath

**Tier 0 → Assembler →** build **Ironworks (A)** + the **★ Smart Plating** station
→ deliver **Phase 1** (50 Smart Plating) → build **Copperworks (B)** then
**Steelworks (C)** + **★ Versatile Framework / Automated Wiring** → **Phase 2** →
**Oil & Electronics (D)** + **★ Modular Engine / ACU** → **Phase 3** → train out
to **Aluminium (E)** and **Nuclear (F)** + **★ ADS / MFG / TPR / Nuclear Pasta** →
**Phase 4** → **Quantum & SAM (G)** + **★ Biochemical Sculptor / AI Expansion
Server / Ballistic Warp Drive** → **Phase 5** → **launch 🚀**.

## How each district is built

1. **Lay the platform** (above ground, near the right ore) — sized with room to
   grow. You don't build the whole district at once.
2. **Build it area-by-area** in the order the app lists (raw/ingot end first,
   finished end last), each area only when its inputs are flowing.
3. **Manifold** each bank: feed/collect from both sides; split a trunk into
   parallel belts when its rate tops one belt (the belt web shows this). Fluids on
   pipes.
4. **Expand it each phase** as later consumers come online (the app's *Expand …*
   steps tell you how many machines to add).
5. **MAX**: overclock every machine to 250% (3 Power Shards) + a Somersloop (2×) —
   ~⅕ the machines. Toggle ⚡ MAX off in Blueprints for a 100% plan.

## Don't forget

- **Power** — switch biomass → coal → fuel → nuclear ahead of demand (see
  `power.md` and the app's power plan). Overclocking is power-hungry; ramp it.
- **Utilities** — pump **water**; handle **Heavy Oil Residue** (Residual Fuel or
  sink); store/process **Nuclear Waste**. The app flags these per district.
- **Transport** — belts local, **trains** to E/F/G, **drones** for small high-value
  parcels (see `train-network.md`).
